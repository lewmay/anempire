# Video Assets

Place your transition video file here as `transition.mp4`.

The video will play full-screen between the quiz completion and results page.

## Requirements:
- Format: MP4
- Recommended resolution: 1920x1080 or higher
- The video should be optimized for web (H.264 codec)
- Keep file size reasonable for fast loading

## Video Behavior:
- Preloads completely before playback
- Plays once, full-screen
- Disables all user interactions during playback
- Navigates to results immediately when finished
